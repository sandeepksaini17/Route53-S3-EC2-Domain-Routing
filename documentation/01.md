# Step 1 – Launch Amazon EC2 Instance

## Objective

Launch an Amazon EC2 instance to host the web application.

## Steps

1. Open AWS Console.
2. Navigate to EC2 Dashboard.
3. Click **Launch Instance**.
4. Enter the instance name.
5. Select **Amazon Linux 2023** AMI.
6. Choose **t2.micro** instance type.
7. Create or select an existing Key Pair.
8. Configure the Security Group:
   - SSH (22)
   - HTTP (80)
   - HTTPS (443)
9. Launch the instance.

## Install Apache

```bash
sudo yum update -y
sudo yum install httpd -y
sudo systemctl start httpd
sudo systemctl enable httpd
```

## Verification

Open the EC2 Public IP in a browser and verify that the web page loads successfully.
