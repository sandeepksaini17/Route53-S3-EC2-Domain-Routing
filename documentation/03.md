# Step 3 – Configure Amazon Route 53

## Objective

Route custom domain names to Amazon S3 and Amazon EC2.

## Steps

1. Open Amazon Route 53.
2. Create or open the Hosted Zone.
3. Create an Alias A Record:

```
www.sandeepaizone.online
```

Target:

```
Amazon S3 Website Endpoint
```

4. Create another A Record:

```
app.sandeepaizone.online
```

Target:

```
Elastic IP of EC2
```

5. Save the records.

## DNS Records

| Record | Target |
|----------|--------|
| www | Amazon S3 |
| app | Amazon EC2 |
