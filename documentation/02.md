# Step 2 – Configure Amazon S3 Static Website

## Objective

Host a static website using Amazon S3.

## Steps

1. Open Amazon S3.
2. Create a bucket.
3. Bucket name:

```
www.sandeepaizone.online
```

4. Disable Block Public Access.
5. Upload `index.html`.
6. Enable Static Website Hosting.
7. Set:

```
Index Document
index.html
```

8. Add a Bucket Policy to allow public read access.

## Verification

Open the S3 Website Endpoint and verify that the static website is displayed.
