# Step 4 – Verification

## Objective

Verify that DNS routing is working correctly.

## Test

### Static Website

```
https://www.sandeepaizone.online
```

Expected Result

The Amazon S3 static website loads successfully.

---

### EC2 Website

```
https://app.sandeepaizone.online
```

Expected Result

The application hosted on the Amazon EC2 instance loads successfully.

---

## Final Architecture

```text
               Internet
                    │
                    ▼
             Amazon Route 53
            /                \
           ▼                  ▼
www.sandeepaizone.online   app.sandeepaizone.online
          │                        │
          ▼                        ▼
      Amazon S3                Amazon EC2
```

---

## Result

Successfully configured Amazon Route 53 to route:

- `www.sandeepaizone.online` → Amazon S3 Static Website
- `app.sandeepaizone.online` → Amazon EC2 Application
